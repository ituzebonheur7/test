> **"Ituze"** (Kinyarwanda: *Peace/Calm*) + **"Bonheur"** (French: *Happiness*)

**Ituze Bonheur** is a digital sanctuary designed to promote mental wellness and emotional balance, specifically tailored for the Rwandan community and beyond. This repository contains the source code for the landing page and the interactive games hub, blending a high-tech "Matrix" aesthetic with therapeutic wellness resources.

🔗 **Main Site:** [ituzebonheur.com](https://ituzebonheur.com/)  
🎮 **Games Hub:** [ituzebonheur.com/games](https://ituzebonheur.com/games)

---

## 🚀 Project Overview

The **Matrix-Homepage** serves as the digital front door for a modern approach to mental health. By using gamification and a tech-forward interface, we make mental health resources accessible, engaging, and less intimidating for the digital generation.

### Key Features:
* **Digital Sanctuary:** A calming user interface designed to reduce stress and cognitive load.
* **Wellness Games:** A curated selection of games (found at `/games`) designed for stress relief, focus, and mindfulness.
* **The "Matrix" Aesthetic:** A unique tech-inspired visual language representing the "reprogramming" of negative thought patterns.
* **Booking Integration:** Seamless connection to mental health professionals and wellness sessions.

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).
* **Animations:** Canvas API (Matrix Rain effect), CSS Keyframes.
* **Optimization:** Mobile-first responsive design for accessibility on all devices.
* **Deployment:** Optimized for hosting on GitHub Pages, Netlify, or Vercel.

---

## 🎮 The Games Hub (`/games`)

Our games are interactive tools for mindfulness and cognitive grounding:
1.  **Breath Control:** Sync your breathing with visual code pulses to regulate the nervous system.
2.  **Focus Matrix:** Cognitive puzzles to help ground users during moments of high anxiety.
3.  **Positive Logic:** Rewiring internal dialogues through interactive choice-based scenarios.

## 💻 Local Development

To run this project locally and contribute:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/ituzebonheur7/Matrix-Homepage.git](https://github.com/ituzebonheur7/Matrix-Homepage.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd Matrix-Homepage
    ```
3.  **Launch:**
    Simply open `index.html` in your preferred browser, or use a local development server like **Live Server** in VS Code.
